# Empirical Validation Plan

## Objective

Turn the entropy-gated conductance equation into a falsifiable standalone model. The immediate target is not a full device demonstration. The immediate target is to verify that BZ phase lifting, slip counting, entropy production, adaptive damping, and the new topological readout interact the way the boxed law says they should.

## Current Numerical Scope

The committed reference simulation in this repository now covers two layers:

- a synthetic BZ bundle that supplies current magnitudes, raw phases, and a time-dependent `pi_a(k,t)` threshold,
- an entropy-gated conductance update driven by a BZ-averaged lifted phase,
- and an effective FHS/QWZ readout layer that maps the trajectory into a time-resolved effective mass, spectral gap, and Chern number.

That is enough to test the core adaptive mechanism before attaching a full admittance-network forward model or a hardware-specific topological readout.

## Core Predictions

1. The conductance phase should relax toward the BZ-resolved lifted phase whenever `mu_G(S) > 0`.
2. A localized slip burst should create discrete entropy injections through `kappa sum |Delta w|`.
3. As entropy rises, `alpha_G(S)` should fall and `mu_G(S)` should rise.
4. The full entropy-gated law should suppress burst-time conductance overshoot relative to a frozen-rate baseline.
5. The BZ proxy `epsilon_eff` derived from `mean_BZ(1 / pi_a)` should remain bounded and smooth.
6. The FHS/QWZ readout should stay numerically stable and report a reproducible topological summary along the same trajectory.

## Implemented Phase 1: Synthetic BZ Reference Case

The current repo implements this phase in [simulations/run_reference_simulation.py](../simulations/run_reference_simulation.py).

- Sample a one-dimensional BZ with a synthetic current bundle.
- Drive a localized slip burst that forces integer sheet jumps.
- Compare a fixed-rate baseline against the full entropy-gated law.
- Record conductance magnitude, entropy, slip counts, and `epsilon_eff`.

## Next Validation Phases

### Phase 2: Network Forward Model

- Replace the synthetic BZ bundle with a small admittance-network forward solve.
- Let edge currents be generated by the evolving complex conductance state instead of an imposed bundle.
- Measure phase-lock quality, entropy budgets, and settling times under controlled disorder.

### Phase 3: Topological Readout Layer

- Attach the BZ ruler output to a concrete effective Hamiltonian or circuit Laplacian.
- Compute a discrete topological invariant with an FHS-style pipeline on successive mesh refinements.
- Test whether adaptive entropy gating changes access to the topological regime, not just local conductance transients.

This phase is now implemented at the readout level with an effective QWZ Hamiltonian and an FHS Chern-number computation. The current committed artifacts are [data/topological_readout_trace.csv](../data/topological_readout_trace.csv), [data/topological_readout_summary.json](../data/topological_readout_summary.json), and [images/topological_readout.svg](../images/topological_readout.svg).

### Phase 4: Hardware Path

Two practical hardware paths are natural.

1. Topolectrical circuits: programmable complex admittances and impedance-based edge-state readout.
2. Photonics: tunable complex couplings and interferometric phase reconstruction.

The right first experiment is whichever platform can measure a resolved complex phase history and implement a controllable slip threshold.

## Minimal Falsifiers

The law should be treated as weakened if any of the following happen consistently.

1. The lifted phase history does not produce coherent conductance phase alignment.
2. Slip events fail to correlate with entropy spikes.
3. Raising entropy does not reduce `alpha_G` or increase `mu_G`.
4. The entropy-gated law cannot reduce burst-time conductance overshoot relative to a frozen-rate baseline.
5. The BZ proxy quantities become numerically unstable in otherwise smooth runs.
6. The topological readout changes unpredictably under mesh refinement or between repeated runs of the same trajectory.

## Near-Term Extensions
